# Roman Numeral Converter

##### Created at Epicodus with pair programming

## Description

Converts a number input into a roman numeral

## Specifications
|Step | input | output|
|-----:|:-----:|:----:|
|Convert 1-3 to I-III| 2| II|
|Convert 4-8 to IV-VIII | 7| VII|
|Convert 9-13 to IX-XIII | 12 | XII|

## Technologies Used

* HTML, CSS, JavaScript
